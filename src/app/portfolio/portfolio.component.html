NextConnect

NextConnect is a web application I wrote while working at Nucor Corporation. While working on integrating a wholly owned subsidiary's data into our Oracle E-Business Suite instance we determined it would be great to have a nice, friendly user interface for our end users. I set to work developing the UI and working with the subsidiary company that was providing the data. Once the application was deployed to production our users could log into NextConnect to view purchase orders, ASNs (Advanced Shipping Notices), receipts, invoices and other information.

What Makes It Work?

NextConnect is almost entirely written in Angular with TypeScript. Backend data is provided via a JSON RESTful API hosted on Azure. The open source UI framework PrimeNG was used extensively and features it's premium template, Ultima which utilizes responsive Material Design, HTML5 and CSS3.
